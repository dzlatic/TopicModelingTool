APIs:

/models Returns JSON with all Models in the system.
/model/<int: model_id> Returns JSON with details about Model specified with model_id.
Runs the inference operation using Model specified model_id over the text defined in body with 'Content-Type' = 'text/plain;charset=UTF-8', saves the text and results into Inference object, that is then returned in JSON.
/model/<int: model_id>/inferences Returns JSON with details about all Inferences for the Model specified with model_id.
/model/<int: model_id>/inference/<int: inference_id> Returns JSON with details about Inference specified with inference_id of the Model specified with model_id.
Delete Inference specified with inference_id of the Model specified with model_id and returns confirmation as JSON.
/model/<int: model_id>/topic/<int: topic_number> Returns JSON with details about Topic specified with topic_number of the model specified with model_id.
Updates Topic specified with topic_number of the model specified with model_id. Sets alias and/or action attributes specified in JSON body {"alias":"...", "action":"..."} and returns updated Topic as JSON.

ADMIN:

/model/<int: model_id>/edit_topics UI that allows looking into topic details and editing topic alias and action attributes.
